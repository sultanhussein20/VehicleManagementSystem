# Vehicle Management System 🚗🏍️

مشروع تجريبي بلغة C# يوضح مبادئ **البرمجة الكائنية (OOP)** مثل:
- الوراثة (Inheritance)
- إعادة تعريف الدوال (Method Overriding)
- إنشاء الكائنات (Object Instantiation)

## 📂 المكونات
- **Car.cs**: يمثل سيارة ويحسب استهلاك الوقود.
- **Motorcycle.cs**: يمثل دراجة نارية.
- **Program.cs**: نقطة البداية للتطبيق.

## ⚙️ كيفية التشغيل
1. افتح المشروع في Visual Studio.
2. شغّل البرنامج (Ctrl + F5).
3. راقب المخرجات في الـ Console.

## 🎯 الهدف
تعلم أساسيات OOP في C# من خلال مثال عملي لإدارة المركبات.
